## TEMPLATE 1: BOLD STATEMENT (Hook Slide)
**Use for:** Slide 1 of every carousel

### Step-by-Step Build:

**1. Background (2 min)**
- Click background area
- Click color picker -> Choose **pure black (#000000)**

**2. Add Main Text (5 min)**
- Press **"T"** or click **"Text"** in left sidebar
- Click **"Heading"**
- Type your hook: "YOUR BOLD HOOK GOES HERE"
- Select all text (Ctrl/Cmd + A)
- **Font:** Search and select **"Bebas Neue"** or **"Anton"**
- **Size:** 110pt (adjust between 90-120pt based on text length)
- **Color:** Pure white (#FFFFFF)
- **Alignment:** Center
- **Text case:** ALL CAPS (type in all caps or use text transform)

**3. Position Text (1 min)**
- Select text box
- Drag to center of canvas
- Or use alignment tools:
  - Top menu -> Position -> Center horizontally
  - Top menu -> Position -> Center vertically

**4. Add Logo (2 min)**
- Click **"Uploads"** in left sidebar
- Upload your DS Marketing logo (white version)
- Drag logo to canvas
- **Position:** Bottom right corner
- **Size:** Resize to about 15% of canvas width (~160px wide)
- Leave 50px margin from bottom and right edges

**5. Save Template**
- Click **"File"** -> **"Save"**
- Name: "DS Template 1 - Bold Statement"
- Move to your templates folder

### Quick Duplication Hack:
- Right-click the page -> **"Duplicate page"** 9 times
- Now you have 10 slides ready
- You'll only edit the text on slide 1; keep rest for other templates

## TEMPLATE 2: NUMBERED POINT (Content Slides - White Background)
**Use for:** Slides 2, 4, 6, 8 (even numbers)

### Step-by-Step Build:

**1. Background**
- Background color: **Pure white (#FFFFFF)**

**2. Add Large Number (Top Left)**
- Add text box
- Type: "01"
- Font: **Bebas Neue**
- Size: **140pt**
- Color: **Black (#000000)**
- Position: Top left, with margins
  - 100px from left edge
  - 80px from top edge

**3. Add Main Point Text**
- Add text box below number
- Type: "YOUR MAIN POINT GOES HERE IN 2-3 LINES MAX"
- Font: **Montserrat Bold**
- Size: **60pt**
- Color: **Black**
- Width: 80% of canvas
- Position:
  - 100px from left edge (aligned with number)
  - 250px from top
- Line height: **1.4** (go to text spacing if needed)

**4. Add Supporting Text (Optional)**
- Add smaller text box below main point
- Type: "Optional supporting detail or explanation"
- Font: **Montserrat Regular**
- Size: **38pt**
- Color: **Dark gray (#1A1A1A)**
- Same left alignment as main point
- 40px below main point

**5. Add Slide Number**
- Add text box in bottom right
- Type: "2/10"
- Font: **Montserrat Light**
- Size: **28pt**
- Color: **Light gray (#808080)**
- Position: Bottom right
  - 60px from right edge
  - 50px from bottom edge

**6. Save Template**
- Name: "DS Template 2 - Numbered Point (White)"
- Save to templates folder

## CREATING VARIATIONS (Advanced)

### Variation A: Add Thin Divider Lines

**For Templates 2 & 3:**
1. Click **"Elements"** -> Search **"line"**
2. Choose a simple straight line
3. Color: Black (on white slides) or White (on black slides)
4. Width: 300px
5. Height: 3px
6. Position: Between number and main text

### Variation B: Circled Numbers

**Instead of plain numbers:**
1. Add your number text (01, 02, etc.)
2. Click **"Elements"** -> Search **"circle outline"**
3. Choose a circle with stroke only (no fill)
4. Color the stroke: Black or White
5. Stroke width: 6px
6. Position circle around the number
7. Group together (Ctrl/Cmd + G)

### Variation C: Quote Style (for client testimonials)

**Build on Template 2:**
1. Add large quotation marks: " "
2. Font: Bebas Neue, 200pt, light gray
3. Position behind/around main text
4. Set opacity to 15%
5. Add client name in small text below quote

## TIME-SAVING HACKS

### Hack 1: Master Template Document
**Instead of creating templates separately:**
1. Create ONE document with all 5 templates as pages 1-5
2. When creating carousels, duplicate page 1 (hook), then add pages 2-5 as needed
3. This saves you from switching between files

### Hack 2: Text Presets
1. Create perfect text box with font, size, color
2. Click on text box -> top menu -> **"Add to designs"** (or "Save style")
3. Reuse on every slide

### Hack 3: Auto-Replace Numbers
1. Create slides 2-8 with "XX" instead of numbers
2. Use Find & Replace (some Canva plans) or manually update
3. Way faster than retyping

### Hack 4: Color Picker Shortcut
1. Click eyedropper tool when changing colors
2. Click your logo to grab exact black/white
3. Ensures perfect color matching

## COMMON MISTAKES & FIXES

**Mistake: Text is too small**
-> Fix: Never go below 40pt on any slide

**Mistake: Slides look too similar**
-> Fix: Alternate between black and white backgrounds (Templates 2 & 3)

**Mistake: Too much text on one slide**
-> Fix: Split into 2 slides, use supporting text sparingly

**Mistake: Logo too big**
-> Fix: Logo should be 10-15% of slide width, not bigger

**Mistake: Inconsistent margins**
-> Fix: Use Canva rulers (View -> Show rulers) and snap to grid

**Mistake: Low contrast**
-> Fix: Only use pure black (#000000) and pure white (#FFFFFF), no in-betweens
