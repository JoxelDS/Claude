## HOW TO USE THESE LINKS

### Option A: Browser (One by One)
1. Copy a link from above
2. Paste it in your browser address bar
3. Wait 10-30 seconds for the image to generate
4. Right-click the image -> "Save image as..."
5. Name it according to the slide (e.g., `slide_01_hook.png`)
6. Repeat for all 23 images + 3 reels

### Option B: Run the Script (All at Once)
1. Download `generate_with_ai.py` from this repo
2. Open Terminal on your computer
3. Run: `python3 generate_with_ai.py`
4. All 23 images + 3 reels generate automatically
5. Find them in `./ai-output/images/` and `./ai-output/reels/`

### Option C: Open All Tabs at Once
1. Download `open_all_links.py` from this repo
2. Run: `python3 open_all_links.py`
3. All 26 links open in your browser tabs
4. Save each image as it loads
