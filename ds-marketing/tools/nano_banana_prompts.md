## TIPS FOR BEST RESULTS IN NANO BANANA PRO

1. **Generate at 1080x1080** (square) for Instagram
2. **If text looks wrong:** Generate without text, then add text in Canva using Bebas Neue font
3. **For consistency:** Generate all slides in one session so the style stays uniform
4. **Regenerate** any slide that doesn't match the dark cinematic aesthetic
5. **Best approach:** Generate the backgrounds/atmosphere with Nano Banana, then overlay your text in Canva for perfect typography control

## HYBRID APPROACH (Recommended)

For the best results, combine AI + Canva:

1. Use Nano Banana to generate **atmospheric dark backgrounds** with this prompt:
```
Premium dark cinematic background texture, 1080x1080 square. Pure black with subtle atmospheric fog, smoke wisps, and dramatic spotlight from above. Moody luxury editorial aesthetic. No text, no objects, just atmospheric dark background with beautiful lighting. Ultra high quality, 4K.
```

2. Generate 3-4 variations of dark backgrounds
3. Import them into **Canva** as background images
4. Add your text using **Bebas Neue** and **Montserrat** fonts
5. This gives you perfect typography + stunning AI-generated atmospherics
